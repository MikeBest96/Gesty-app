# GymTracker

GymTracker is a mobile app for tracking workouts, measuring gym progress, exploring a training library, and getting AI-guided coaching support.

## Stack

- Expo
- React Native
- TypeScript
- Expo Router

## Current status

The app scaffold is initialized and the default Expo tabs have been replaced with GymTracker product tabs:

- Home
- Log Workout
- Progress
- Library
- Coach

Planning and architecture docs live in [docs](C:\Users\alfbe\OneDrive\Escritorio\Programming for  Dummies\GymTracker\docs).

## Commands

Use these from the project root:

```bash
npm run start
npm run android
npm run web
npm run typecheck
```

If PowerShell blocks `npm`, use `npm.cmd` instead.
